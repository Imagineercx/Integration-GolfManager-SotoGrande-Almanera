# Project Setup

To install the necessary dependencies, run the following commands:

```sh
npm install mysql2
```

## Running Tests

To install the testing dependencies, run:

```sh
npm install --save-dev jest
npm install --save-dev nock
```

To execute the tests, use the following command:

```sh
npm test
```
